= AsciiDoc DOCX

The `asciidoc-docx` module provides bidirectional integration between AsciiDoc and Microsoft Word (DOCX) formats.
It leverages Docx4J to map AsciiDoc structures to WordprocessingML and vice versa.

== Key Features

* **AsciiDoc to DOCX**: `AsciiDocToDocx` converts the AsciiDoc model to Docx4J `R` (Run) objects, preserving formatting.
* **DOCX to AsciiDoc**: `DocxToAsciiDoc` converts Docx4J objects or full documents back to AsciiDoc text.
* **Formatting Mapping**: Handles complex mapping of styles, bold, italic, and other inline formatting between the two formats.

== Quick Start

[source,java]
----
AsciiDocModel model = ...;
AsciiDocToDocx converter = new AsciiDocToDocx();
List<Object> runs = converter.convert(model);
----
