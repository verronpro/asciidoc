= Office-stamper AsciiDoc Compiler

The `asciidoc-compiler` module provides a high-level facade for all AsciiDoc operations.
It includes PNG export utilities powered by Apache Batik, allowing for easy conversion of AsciiDoc content to image formats.

== Key Features

* **Unified API**: `AsciiDocCompiler` provides a single entry point for common operations.
* **Image Export**: Built-in support for compiling AsciiDoc directly to SVG or PNG.
* **Customization**: Support for different DPI settings and themes.
* **Integration**: Combines functionality from core, converters, and docx modules.

== Quick Start

[source,java]
----
AsciiDocCompiler compiler = new AsciiDocCompiler();
Path pngPath = compiler.compileToPng("source.adoc", "output.png", 300, Theme.WORD);
----

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.office-stamper</groupId>
    <artifactId>asciidoc-compiler</artifactId>
    <version>${project.version}</version>
</dependency>
----
