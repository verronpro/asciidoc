= AsciiDoc Preview Extension

The `asciidoc-preview` module provides an AsciidoctorJ block macro extension to embed editor-simulated previews of AsciiDoc templates into documentation.

== Key Features

* **`preview::[]` Macro**: Automatically renders and embeds PNG previews of AsciiDoc files.
* **Theming**: Supports `theme`, `format`, and `dpi` attributes for consistent previews.
* **Caching**: Intelligent caching system that only regenerates images when the source file changes.
* **Easy Integration**: SPI discovery for automatic registration with AsciidoctorJ.

== Usage in AsciiDoc

[source,asciidoc]
----
preview::my-template.adoc[theme=word, dpi=300]
----

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.asciidoc</groupId>
    <artifactId>asciidoc-preview</artifactId>
    <version>${project.version}</version>
</dependency>
----
