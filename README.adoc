= Office-stamper AsciiDoc Modules

This project provides a suite of modules for handling AsciiDoc content within the Office-stamper ecosystem.
It enables bidirectional conversion between AsciiDoc and DOCX, as well as high-quality rendering to HTML, SVG, and PNG.

== Modules

* link:asciidoc-core/README.adoc[asciidoc-core]: Foundation data model and parsing logic.
* link:asciidoc-converters/README.adoc[asciidoc-converters]: Lightweight rendering to HTML, SVG, and Text.
* link:asciidoc-docx/README.adoc[asciidoc-docx]: Conversion between AsciiDoc and Microsoft Word (DOCX).
* link:asciidoc-compiler/README.adoc[asciidoc-compiler]: High-level API and PNG export utilities.
* link:asciidoc-preview/README.adoc[asciidoc-preview]: AsciidoctorJ extension for embedding previews in documentation.
