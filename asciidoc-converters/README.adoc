= Office-stamper AsciiDoc Converters

The `asciidoc-converters` module provides lightweight converters from the `AsciiDocModel` to various text-based and vector formats.

== Key Features

* **HTML Converter**: `AsciiDocToHtml` converts the model to basic HTML spans.
* **Text Converter**: `AsciiDocToText` converts the model to plain text.
* **SVG Converter**: `AsciiDocToSvg` converts the model to SVG, with support for different editor themes (Word, Google Docs, LibreOffice).

== Quick Start

[source,java]
----
AsciiDocModel model = ...;
AsciiDocToHtml converter = new AsciiDocToHtml();
String html = converter.convert(model);
----

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.office-stamper</groupId>
    <artifactId>asciidoc-converters</artifactId>
    <version>${project.version}</version>
</dependency>
----
